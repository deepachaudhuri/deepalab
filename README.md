# DeepaLab — AWS Lab

This repository contains materials and exercises for an AWS lab.

Purpose
- This project is created for an AWS lab to practice creating and managing AWS resources.

Contents
- `lab1/` — Example CloudFormation template and lab materials.

Prerequisites
- An AWS account
- AWS CLI configured (optional)

How to use
- Review the files under the `lab1` directory to get started with the lab exercises.

Contact
- For questions, update the README or open an issue in this repo.
